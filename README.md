# Google Map APIを用いた写真スポット提案システムの開発

## 概要
- GoogleMapAPIを用いた写真スポットを提案するシステム
- 写真を⽤いたSNSなどでの発信によって⼈々に興味を持たせることができる
- 訪問者が旅の思い出として美しい写真を撮ることができる
- RTでGoogle Mapの使⽤を可能にした

## システム概要
- GoogleMapAPIを用いた座標と航空写真の取得
- GoogleMapへのマーカー入力
- csv出力
